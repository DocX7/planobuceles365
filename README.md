# Família na Palavra

Aplicativo simples para GitHub Pages com plano anual de leitura bíblica em ordem cronológica e uma área infantil para Arthur.

## O que tem

- Leitura diária cronológica em 365 dias.
- Perfis Marcus e Ingrid com progresso separado.
- Progresso salvo no navegador pelo `localStorage`.
- Área do Arthur com historinha bíblica, versículo, oração e pergunta.
- PWA básico para instalar no celular.

## Publicar no GitHub Pages

1. Suba todos os arquivos na raiz do repositório.
2. Acesse `Settings > Pages`.
3. Escolha `Deploy from a branch`.
4. Selecione `main` e `/root`.
5. Salve e aguarde o link do GitHub Pages.

## Observação

Esta versão mostra as referências da leitura diária. Ela não inclui o texto bíblico completo para manter o app leve e simples.
